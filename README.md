# Agricultural Analysis System 🌱

A comprehensive web application for agricultural analysis using Sentinel-2 satellite imagery and machine learning. Perform crop type mapping, health monitoring, yield estimation, and soil analysis.

![Dashboard Preview](screenshots/dashboard.png)

## 🚀 Features

- **Crop Analysis**: Identify crop types, monitor health, detect stress
- **Yield Estimation**: Machine learning-based yield predictions
- **Soil Analysis**: Soil type mapping, mineral content, moisture estimation
- **Time Series Analysis**: Phenological metrics and seasonal patterns
- **Interactive Maps**: Draw AOI, visualize results
- **Export Capabilities**: Download reports in multiple formats

## 🛠️ Technology Stack

- **Backend**: Flask, Earth Engine Python API
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Visualization**: Leaflet, Chart.js
- **Machine Learning**: Scikit-learn, Random Forest
- **Deployment**: Render (PaaS)

## 📋 Prerequisites

- Python 3.8+
- Google Earth Engine account
- Google Cloud Project with Earth Engine API enabled
